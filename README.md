# boxify
Python function that returns a string with each character inside an ASCII "box".  The function is tweetable, being only 91 chars.

Only tested on Python 3.4.

### Usage
```python
from boxify import boxify
print(boxify('Python'))
```
Which will output
```
+---+---+---+---+---+---+
| P | y | t | h | o | n |
+---+---+---+---+---+---+
```
